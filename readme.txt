[概要]
プレイヤーの磁力を切り替えることで発生する引き寄せや反発を利用してゴールを目指すゲームです。

ターゲット：走り抜ける爽快感を楽しみたい10代
コンセプト：磁力による動きの変化の楽しさを味わえるゲーム

[実装内容]
α版（～1/26）
・磁石
磁力の発するオブジェクト。プレイヤーが近づくと磁力の影響を受けます。
※現時点ではオブジェクトに近づいても反発しかしません。

β版（～2/2）
・磁石切り替え
磁力を切り替えます。プレイヤーの色が変わり、近くのオブジェクトとプレイヤーの色が同じか違うかでプレイヤーの挙動が変わります。
※現時点では磁力を切り替えても引き寄せはしません。
